# Fan-Page

Goal: Build first HTML featuring favorite musician fan page

Actions:
1. Implement proper HTML page structure
2. Include an image, biography section with a paragraph 
3. Insert a link to another website
4. Create an unordered lists 
5. Use h1 and h2 tags for headings
6. Input a music video of the musician playing a song
7. Proper indentation
8. Create a README.md
